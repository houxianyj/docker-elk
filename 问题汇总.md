### docker-elk启动

```shell
#初始化配置 
docker compose up setup

#启动
docker compose up
```


### kibana 修改中文
```shell
在kibana.yml配置文件中添加一行配置

i18n.locale: "zh-CN"
```



### mysql 数据同步

```shell
docker run -v "/root/es/connectors:/config" --tty --rm docker.elastic.co/enterprise-search/elastic-connectors:8.16.0 /app/bin/elastic-ingest -c /config/config.yml
```


### 查看远程分支

git remote -v
origin  git@github.com:houxianyj/docker-elk.git (fetch)
origin  git@github.com:houxianyj/docker-elk.git (push)
source  https://github.com/deviantony/docker-elk (fetch)
source  https://github.com/deviantony/docker-elk (push)

### 获取远程 source 最新改变
```gitexclude
git fetch source
```

### 确保你在 main 分支
```gitexclude
git checkout main
```

### 合并 source/amin 到你本地 main
```gitexclude
git merge source/main
```

### 推送 main 到自己的origin
```gitexclude
git push origin main
```
